  <h1 align='center'>Guard API</h1>
  <h3 align='center'>Access Control API for Guard</h3>
  <p align='center'> GuardAPI allows users to provide sensitive data to Cosmos users, CosmWasm smart contract administrators and Cosmos protocols/applications without compromising their security. Users can give lessees access to data such as shipping addresses, emails and phone numbers in a manner similar to OAuth and are able to revoke access to their data at any time. </p>
